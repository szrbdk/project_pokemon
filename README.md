# project_pokemon

A simple Pokédex application using Poke API and Flutter.


#### Features

- [x] List pokémons using the API 
- [x] Show pokémon details
- [ ] Catch pokémons and keep the déx record
- [ ] Release pokémons 
- [ ] List caught pokémons 
- [ ] Add/remove favorite caught pokémons 
- [ ] Dark theme
- [x] Language support
- [ ] Test Case

#### API

https://pokeapi.co/
